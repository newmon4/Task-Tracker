# A simple web-based task tracker built with ASP.NET

## Overview
This project is a simple web application with frontend-backend integration. It provides task management functionality through an API built with ASP.NET Core, allowing users to create and manage tasks.

## Key Features
-Create Task
-Delete Task
-Dynamic task list rendering
-Experimental custom API endpoint selection

## Technology Stack
Backend: C# ASP.NET Core
Frontend: vanilla HTML, Javascipt
Languages: C#, Javascript
